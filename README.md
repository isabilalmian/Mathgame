# Maths Game

A browser-based arithmetic quiz game for ages 8–60. Content is aligned to the **Berlin school curriculum** (Berliner Rahmenlehrpläne) and age.

- **Age 8–9:** Grade 4 mathschoolbook  
- **Age 10–11:** Grade 5 mathschoolbook  
- **Age 11–12:** Grade 6 mathschoolbook  
- **Age 12–13:** Grade 7 mathschoolbook  
- **Age 13–14:** Grade 8 … up to **Age 17–18:** Grade 12  
- **Age 19+:** University-level maths  
- Timer per question (max 3 min), lives, scoreboard with rankings  
- Emoji reactions, fireworks when you hit #1, confetti when you win  

Open `index.html` in a browser to play. No build step required.
